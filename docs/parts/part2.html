IOS104 Add a launch screen and app icons

Exercise 2: Create a responsive launch screen

This exercise will teach you how to use the designer to set up a responsive iOS app launch screen for a variety of screen sizes.

Your finished launch screen should look like this:

To complete the exercise, you will need Visual Studio for Windows or macOS with the Xamarin development tools installed. You will also need either an emulator/simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Open the starter solution

This exercise is a continuation of the previous exercise. You can use your existing solution or begin from the prior Exercise 1 > Completed solution in your copy of the cloned or downloaded course materials.

Create a new storyboard

While the template we used to generate this app will include a default LaunchScreen.storyboard, you can add a new storyboard to any existing iOS project.

  1. In the Solution Explorer, right-click the iOS app project node and choose Add > New Item….
  2. Choose the Apple > User Interface > Empty Storyboard template from the Add New Item window.
  3. Give the storyboard a name and click Add.
  1. In the Solution pad, right-click the iOS app node and choose Add > New File….
  2. Select the Storyboard template from the iOS category.
  3. Give your storyboard a name and click New.

Add launch graphic image set

While the starter project already includes an image set in the asset catalog that we will use for this exercise, you can also add a new asset catalog to an asset catalog. These steps are optional for this exercise.

To make an image easily available for a launch storyboard, we add it as an image set of an asset catalog. Feel free to use any image you want, but a launch image is also provided for three densities in the Exercise 2 > Assets solution from your copy of the cloned or downloaded course materials.

  1. From the Solution Explorer, expand the Asset Catalogs node and open the Assets asset catalog.
  2. In the left-hand pane, click the plus (+) icon to add a new image set and name it "LaunchImage".
  3. Click the … icon on the 1x image to pick an image file.
  4. In the file open dialog, select the Assets > launch-icon@1x.png icon and click Open.
  5. Do the same above steps for 2x and 3x images, matching the requested density with the appropriate launch-icon@{scaleFactor}x.png image.
  1. From the Solution pad, open the provided Assets.xcassets asset catalog.
  2. In the left-hand pane, right-click and choose New Image Set.
  3. Select the newly added image set and press F2 to rename it to "LaunchImage".
  4. Inside the new LaunchImage set, click the 1x image to pick an image file.
  5. In the file open dialog, select the Assets > launch-icon@1x.png icon and click Open.
  6. Do the same above steps for 2x and 3x images, matching the requested density with the appropriate launch-icon@{scaleFactor}x.png image.

Add image to storyboard

For this step, we'll create a simple launch screen layout of a centered image with a background color.

  1. In the Solution Explorer, open the provided LaunchScreen.storyboard file.
  2. Inside the Toolbox (View > Toolbox from the Visual Studio menu), find the Image View control and drag it into the view controller.
Note: If you are working from a new empty storyboard, you will have to first drag a View Controller onto the storyboard surface to recieve the image view control.
  1. Select the newly added image view and open the Properties window (View > Properties Window).
  2. Within the Widget tab, select the Image dropdown and choose the LaunchImage image set we added previously.
  3. Switch to the Layout tab and adjust the Width and Height to match your images dimensions: 140 by 140, respectively, for the image provided for this exercise.
  1. From the Solution pad, open the provided LaunchScreen.storyboard file.
  2. Inside the Toolbox pad (View > Pads > Toolbox from the menu), find the Image View control and drag it into the view controller.
Note: If you are working from a new empty storyboard, you will have to first drag a View Controller onto the storyboard surface to recieve the image view control.
  1. Select the newly added image view and open the Properties window (View > Pads > Properties).
  2. Within the Widget tab, select the Image dropdown and choose the LaunchImage image set we added previously.
  3. Switch to the Layout tab and adjust the Width and Height to match your images dimensions: 140 by 140, respectively, for the image provided for this exercise.

Add image constraints

  1. Switch to Constraint Editing Mode by clicking the appropriate button from the right-hand side of the toolbar on the designer window.
  2. To center our image, constrain the center of the image to the center of the view controller.
    1. Drag the image view's center constraint handle, represented by a square in the middle of the control, and drop it on the dashed horizontal center line of the view controller.
    2. Then, drag the same center constraint handle and drop it on the dashed vertical center line of the view controller.
    3. If your image view is offset, double click the affected constraint lines and set the Constant value to 0.

Your image view should be fully constrained at this point by the image dimensions and the center constraints.

Set view controller background color

To give our launch screen's icon a little more impact, lets set a custom background color.

  1. Within the launch storyboard, select the view controller and open the Properties window (View > Properties Window).
  2. Within the Widget tab, select the Custom category of colors in the Background section.
  3. Set the following RGBA (red, green, blue, alpha) values to set it to use a custom blue color:
    • R: 2
    • G: 143
    • B: 255
    • A: 255
  1. Within the launch storyboard, select the view controller and open the Properties window (View > Properties Window).
  2. Within the Widget tab, expand the Background drop-down list and choose the Other… color value.
  3. Inside the color picker window that pops up, choose the Color Sliders tab.
  4. From the input type drop-down list, select RGB Sliders and set the Hex Color # value to 028FFF to use a custom blue color.

Launch the app

To verify the launch screen, run the app on a simulator.

If you miss the launch screen before the app itself runs, you will need to relaunch the app.

Additionally, you can also test the launch screen while the device is in landscape mode or on additional screen sizes like the iPad simulators. It should always be centered, no matter what the screen's dimensions.
