IOS104 Add a launch screen and app icons

Exercise 1: Organize icons in an asset catalog

The primary goal of this lab is to teach you how to set up your iOS app's icons using an asset catalog.

When finished, you will be able to see the app icon in various places within the iOS UI.

To complete the exercise, you will need Visual Studio for Windows or macOS with the Xamarin development tools installed. You will also need either an emulator/simulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Open starter solution

The starter solution contains a sample SavingsPlanner app. We will be adding our icons to this app.

  1. Open the Exercise 1 > Start solution from your copy of the cloned or downloaded course materials in Visual Studio.
  1. Open the Exercise 1 > Start solution from your copy of the cloned or downloaded course materials in Visual Studio for Mac.

Add an asset catalog

While the starter project includes an initial asset catalog, you can also add a new asset catalog to any existing iOS project. These steps are optional for this exercise.

  1. In the Solution Explorer, right-click the Asset Catalogs node and choose Add Asset Catalog.
  2. Leave the Apple > Code > Asset Catalog template selected and give your asset catalog a name.
  3. Click the Add button.
  4. The asset catalog template will give you an app icon image set, but you can add one to an asset catalog by clicking the plus (+) button and choosing App Icon and Launch Image > Add App Icon.
  1. In the Solution pad, right-click the iOS app project node and choose Add > New File….
  2. Select the Asset Catalog template from the iOS category.
  3. Give your new asset catalog a name, or leave the default value, and click New.
  4. To add an app icon image set, right-click in the left side of the asset catalog window, and choose App Icons & Launch Images > New App Icon.
NOTE: Visual Studio will sort your asset catalogs together toward the top of the project rather than with the other project files.

Open the provided asset catalog

Now let's use our own icon for the various app icons in the asset catalog. Feel free to use any custom icons you want, but a set of icons are also provided in the Exercise 1 > Assets solution from your copy of the cloned or downloaded course materials.

  1. From the Solution Explorer, expand the Asset Catalogs node and open the Assets asset catalog.
  2. Select the AppIcon image set from the left-hand list.
  3. Click the … icon on the 40x40 image to pick a new image file.
  4. In the file open dialog, select the Assets > Icon-40.png icon and click Open.
  5. Do the same above steps for the remaining default app icons, matching the requested icon size with the appropriate Icon-{size}.png image.
Note: some icon sizes will be used multiple times to complete the various app icons and their higher-density versions.
  1. From the Solution pad, open the provided Assets.xcassets asset catalog.
  2. Select the AppIcon set from the left-hand side to see the various app icons.
  3. Click the 2x icon for iPhone Notification image to open the file picker.
  4. In the file open dialog, select the Assets > Icon-40.png icon and click Open.
  5. Do the same above steps for the remaining iPhone and iPad app icons, matching the requested icon size with the appropriate Icon-{size}.png image.

Leave the Apple Watch and Mac icons unset, we won't be working with an Apple Watch project or macOS project, but they would be set the same way on this screen.

Note: some icon sizes will be used multiple times to complete the various app icons and their higher-density versions. If you are asked to overwrite an existing icon, click Yes.

Run the app

To verify the new icon is set, run the app on a simulator.

  1. Launch the app in debug mode on any simulator.
  2. After the app launches, hit the home button on the simulator and verify the app icon you set is present on the launcher home screen.
  3. Additionally, you can verify other icon uses. For example, the Spotlight icon can be found by dragging down from the root iOS launcher screen and typing in the name of your app. You can also see the settings icon by scrolling down to the app list in the simulator's Settings app.

Exercise summary

Congratulations! In this lab you configured the icons for an iOS application using an asset catalog.

You can view the completed solution in the Exercise 1 > Completed folder of your copy of the cloned or downloaded course materials.
